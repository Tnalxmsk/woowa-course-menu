## 요구 사항
- 월 ~ 금 점심 식사
  - 월요일에 추천할 카테고리 정하기
  - 각 코치가 먹을 메뉴 추천
  - 화 ~ 금, 위 과정 반복
- 코치 이름 2~4글자
- 코치 2~5명까지
- 못 먹는 음식 0~2개까지 , 구분
  - 없으면 빈 값 입력
- 한 주에 같은 카테고리 최대 2번! 중요중요
- 한 코치에게 중복되지 않은 메뉴 추천! 중요중요
- 잘못된 값 입력 시 재입력 처리
